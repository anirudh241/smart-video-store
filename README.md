# Smart Video Store – Inventory-Aware Recommendation System

## Overview
A full-stack prototype extending the MySQL Sakila database to implement asset health tracking and inventory-aware recommendation filtering. The system includes:
- Extended relational schema for tracks and conditions
- REST API backend (Flask)
- Flutter frontend prototype UI

## Features
- Per-disc inventory & condition tracking
- Ranking & filtering of recommendations
- Backend API
- Flutter UI (demo mode)
